Welcome to my angular demo. It is in angular and shows already a lot of basic concepts
It was build with this command line : $ ng build --deploy-url /releaseDemoAngular4/ --base-href /releaseDemoAngular4/
The angular4 source code project is here : releaseDemoAngular4.zip
  • You can navigate with menu, thanks to router concept
  • Data are given from a fake backend service (json file), using http and Observable (not Promise)
  • Each product image is clickable and lead to product detail page, also using router
  • Star rating composant show how to implement nested component and passing data from parent component and notify parent component from star rating component
  • Project also shows how to create and link modules, how to export Component, directive, or Pipe from a new module to make it visible for other modules
    Notice : les services n'ont pas besoin d'être exporté, ils sont accessibles en injection de dépendance depuis toute l'application (via les contructor())
  • It also embed a login module with fake authentification using httpInterceptor, and a dirty :) logged user panel
  • it shows how to create and use a global variable map, using a simple service
  • Shows concept of routing query parameters, which can be used to memorize previous page parameters in the request
  • Another bonus : ci-dessous une table pas mal, récupérée avec : npm install --save ngx-super-table ; Voir ngx-super-table project on github
